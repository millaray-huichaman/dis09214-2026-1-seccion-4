### ¿Cómo percibimos los colores?

La luz es una onda electromagnética que se ve. Tenemos células conos que detectan las cosas azulosas, verdosas y rojizas.

Un pixel es elemento diseñado ficticio que dentro tiene rojos, verdes y azules.

## Bits

Necesitamos 24 bits para los colores.

si ocupamos 8 bits para cada color, cual es la cantidad de combinaciones que se pueden hacer? -> 256, como el computador comienza a contar desde 0, el máximo valor cuando tenemos 8 bits es 255.

Un bit (dígito binario) es la unidad de información más pequeña utilizada en computación y tecnología digital. El término tiene distintos significados según el contexto en el que se utilice, representa un único valor que puede ser 0 o 1, lo que equivale a un interruptor apagado o encendido, los bits se agrupan para formar datos más complejos.

---
Los bits nos permiten tener una graduación, un intermedio entre encendido y apagado.

> 8 bits = 1 byte
> 24 bits = 3 bytes
